# AI Agents Documentation - User Management Project

This document outlines the AI agents and tools used during the development of the User Management application.

## 1. ChatGPT (Code Generation & Suggestions)
- **Purpose:** Generate boilerplate code for React frontend, Flask API routes, and SQLAlchemy models.
- **Role:** Assisted in writing:
  - React components for User List, Add/Delete user
  - Axios API integration
  - Flask API routes (GET, POST, DELETE)
  - Database setup with SQLAlchemy

## 2. Claude AI (Code Review & Refactoring Suggestions)
- **Purpose:** Reviewed the generated code for:
  - Correctness
  - Clean structure
  - Naming conventions
  - Error handling

## Usage Flow
1. Initial prompts to generate code snippets.
2. AI-generated code reviewed manually by developer.
3. Tested code locally with Flask backend and React frontend.
4. Iterative improvements guided by AI suggestions.
